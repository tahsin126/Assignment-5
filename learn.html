Question-1: What are the different ways to select an element in the DOM?

different ways to select an element in the DOM.
getElementById() Selects element by its ID.
getElementsByClassName() Selects element by class name.
getElementsByTagName() Selects element by tag name.
querySelector()
querySelectorAll() - Selects all elements with same class name.

Question-3: What is event delegation in the DOM?

Event delegation is when to add a single event listener to a parent element instead of multiple listeners. when an event happens on a child element, it bubbles to the parent, and the parent can handle it.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event happen on a child element and then move up to parent elements. it means the event will first happen to child element, and then it will bubble to parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, we use document.createElement().
To add it to the page, we use appendChild().
To remove an element, we use removeChild().
